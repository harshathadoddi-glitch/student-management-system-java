# Student Management System (Java)

## Overview
A console-based Java application to manage student records.  
Implements full CRUD operations using core Java and OOP principles.

## Features
- Add new student
- View all students
- Search by ID
- Update details
- Delete records

## Tech Stack
- Java (JDK 21)
- OOP (classes, objects)
- Collections (ArrayList)
- Scanner (user input)

## Project Structure
- `StudentManagement.java` – main program and menu logic
- `Student` class – model for student data

## How to Run
```bash
javac StudentManagement.java
java StudentManagement
